# Nutrition Tracker App

A full-stack web application for tracking nutrition and personal fitness goals.  
Built with **React**, **Node.js/Express**, **PostgreSQL**, and the **USDA FoodData Central API**.

---

## Features

- **JWT Authentication:** Secure login with role-protected routes
- **Food Database:** Search 350,000+ foods with fast API responses (<400ms)
- **Personalized Macros:** Calculates TDEE using the Mifflin–St Jeor equation
- **Progress Tracking:** Real-time UI updates with React state
- **Secure Backend:** Normalized PostgreSQL schema with 4 tables, 12 REST endpoints, connection pooling, and prepared statements

---

## Tech Stack

- **Frontend:** React, HTML, CSS, Tailwind
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **API:** USDA FoodData Central API
- **Tools:** Git, Docker, Node.js
